# Bhumik | Cyber Security Portfolio

A high-performance, modern cyber security portfolio built with React, Framer Motion, and Three.js.

## 🚀 How to Run

### Method 1: Simple Browser (No Setup)
You can directly open the `index.html` file in any modern web browser.
- Right-click `index.html` -> **Open with** -> **Google Chrome** (or your preferred browser).

### Method 2: Local Development Server (Recommended)
Using a local server ensures all animations and assets (like the background effects) load correctly without browser security restrictions.

#### Option A: Using VS Code Live Server
1. Open this folder in **VS Code**.
2. Install the **Live Server** extension.
3. Click **Go Live** at the bottom right of the editor.

#### Option B: Using Python (If installed)
Run this command in your terminal:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.

#### Option C: Using Node.js (If installed)
Run this command to install a quick server:
```bash
npx serve .
```

## 🛠️ Project Structure

- `index.html`: The main entry point and portfolio UI.
- `myphoto.jpeg`: Profile picture.
- `CV_Bhumik.pdf`: Downloadable resume.
- `assets/`: Project screenshots (e.g., `aws_infra.png`, `keylogger.png`).
- `package.json`: Project dependencies and configuration.

## 💻 Tech Stack
- **React**: Component-based UI.
- **Framer Motion**: Smooth animations and transitions.
- **Three.js (Vanta.js)**: Dynamic animated background.
- **Babel**: In-browser JSX transpilation.
- **CSS3**: Modern glassmorphism and responsive design.
